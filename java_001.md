**Java의 특징**

* Java는 객체지향 프로그래밍(OOP) 언어로, 클래스와 객체를 사용하여 프로그램을 구성한다. 
* JVM(Java Virtual Machine)을 이용하여 운영체제에 관계없이 실행할 수 있는 플랫폼 독립성을 가진다. 
* "한 번 작성하면 어디서나 실행된다(Write Once, Run Anywhere)"는 특징으로 유명하다. 
* 자동 메모리 관리(Garbage Collection), 높은 보안성, 멀티스레드 지원 등의 기능을 제공한다.



**Java의 장점**

* 플랫폼에 독립적이어서 다양한 운영체제에서 실행 가능하다.
* 객체지향 언어로 코드의 재사용과 유지보수가 쉽다.
* 자동 메모리 관리 기능으로 개발자의 부담을 줄여준다.
* 보안성이 높고 안정적인 프로그램 개발이 가능하다.
* 다양한 라이브러리와 개발자 커뮤니티가 잘 구축되어 있다.



**Java의 단점**

* JVM을 사용하기 때문에 C/C++보다 실행 속도가 다소 느릴 수 있다.
* 메모리 사용량이 비교적 많다.
* 문법이 다소 길고 코드 작성량이 많은 편이다.
* 저수준 시스템 제어가 필요한 프로그램 개발에는 적합하지 않다.
